template<typename T, typename Limit = primitive_of_t<T>>
struct boost::boost::dispatch::factory_of< T, Limit >

Recursive meta-model generator.

For any type T, returns a template alias able to reconstruct T using a different primitive type.

Template Parameters
TType to build a type-factory of
Limit
Models:
Semantic:

For any type T:

boost::dispatch::factory_of<T>::apply;

return a template alias so that, for any type U:

using X = apply<U>;

generates a type X so that:

std::is_same<boost::dispatch::primitive_of<X>, U>::value

and for which the boost::dispatch::model_of template aliases are equivalent.

Put in another way, boost::dispatch::factory_of is a recursive application of boost::dispatch::model_of so that every nested type are traversed during reconstruction.
